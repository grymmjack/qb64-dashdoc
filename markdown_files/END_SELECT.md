## END SELECT
---

### The END SELECT statement ends a SELECT CASE statement block.

#### SYNTAX

`END SELECT`

#### DESCRIPTION
* The statement is required to end a [SELECT](./SELECT.md) [CASE](./CASE.md) block or there will be an error .
