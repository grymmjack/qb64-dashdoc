<h1 class="firstHeading mw-first-heading" id="firstHeading">_glIsList</h1>
<div class="vector-body" id="bodyContent">
<div class="mw-body-content mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><div class="mw-parser-output"><p>The <b>_glIsList</b> function tests for display list existence.
</p>
<h2><span class="mw-headline" id="Syntax">Syntax</span></h2>
<dl><dd>GLboolean <i>result</i> = <a class="mw-selflink selflink">_glIsList</a> (GLuint <i>list</i>)</dd></dl>
