# _R2D
> The _R2D function converts a radian value into a degree value.

## SYNTAX
`result! = _R2D ( num )`

## EXAMPLES
> Example: Coverting Radian into Degree.

```vb
INPUT "Give me an angle in Radians ", R
D = _R2D(R)
PRINT "That angle in Degrees is "; D
```
