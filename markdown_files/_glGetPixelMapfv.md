<h1 class="firstHeading mw-first-heading" id="firstHeading">_glGetPixelMapfv</h1>
<div class="vector-body" id="bodyContent">
<div class="mw-body-content mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><div class="mw-parser-output"><p>The <b>_glGetPixelMapfv</b>, <b>_glGetPixelMapuiv</b>, and <b>_glGetPixelMapusv</b> statements return the specified pixel map.
</p>
<h2><span class="mw-headline" id="Syntax">Syntax</span></h2>
<dl><dd><a class="mw-selflink selflink">_glGetPixelMapfv</a> GLenum <i>map</i>, GLfloat <i>*values</i></dd></dl>
<p>
