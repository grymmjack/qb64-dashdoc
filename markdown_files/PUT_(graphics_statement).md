## PUT (graphics statement)
---

### The PUT graphics statement is used to place GET or BSAVE file images stored in the designated array.

#### SYNTAX

`PUT [[[STEP]]] ( column , row ), Array( [ index ] ) [,] [[[_CLIP]]]  [{PSET|PRESET|AND|OR|XOR}]][, omitcolor ]`
