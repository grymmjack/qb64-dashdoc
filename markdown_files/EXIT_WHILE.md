## EXIT WHILE
---

### The EXIT WHILE statement does exit from a WHILE...WEND loop.

#### SYNTAX

`EXIT WHILE`

#### DESCRIPTION
* Use [EXIT](./EXIT.md) [WHILE](./WHILE.md) to immediately exit a [WHILE](./WHILE.md)...[WEND](./WEND.md) loop.
* Execution will proceed after [WEND](./WEND.md) .
