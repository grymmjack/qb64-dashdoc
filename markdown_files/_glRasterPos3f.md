<h1 class="firstHeading mw-first-heading" id="firstHeading">_glRasterPos3f</h1>
<div class="vector-body" id="bodyContent">
<div class="mw-body-content mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><div class="mw-parser-output"><p>The <b>_glRasterPos3f</b> statement specifies the raster position for pixel operations.
</p>
<h2><span class="mw-headline" id="Syntax">Syntax</span></h2>
<dl><dd><a class="mw-selflink selflink">_glRasterPos3f</a> GLfloat <i>x</i>, GLfloat <i>y</i>, GLfloat <i>z</i></dd></dl>
