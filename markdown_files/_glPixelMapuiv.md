<h1 class="firstHeading mw-first-heading" id="firstHeading">_glPixelMapuiv</h1>
<div class="vector-body" id="bodyContent">
<div class="mw-body-content mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><div class="mw-parser-output"><p>The <b>_glPixelMapuiv</b> statement sets up pixel transfer maps.
</p>
<h2><span class="mw-headline" id="Syntax">Syntax</span></h2>
<dl><dd><a class="mw-selflink selflink">_glPixelMapuiv</a> GLenum <i>map</i>, GLsizei <i>mapsize</i>, const GLuint <i>*values</i></dd></dl>
